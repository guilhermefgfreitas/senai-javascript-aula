O que é JavaScript

JS (JavaScript) é uma linguagem de programação leve, interpretada e de alto nível, amplamente utilizada para criar interatividade e dinamismo em páginas web, mas também usada em back-end com ambientes como o Node.js.

Principais características do JavaScript

  1. Simplicidade
  2. Funcionalidade estendida
  3. Popularidade
